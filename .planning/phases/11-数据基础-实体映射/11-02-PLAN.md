for (NovelContinentRegion r : regions) {

String stripped = name.replaceAll("[宗派门殿阁会帮谷山城族教院宫楼庄寨盟]$", "");
for (NovelContinentRegion r : regions) {
    String rStripped = r.getName().replaceAll("[宗派门殿阁会帮谷山城族教院宫楼庄寨盟]$", "");
    if (!stripped.isEmpty() && stripped.equals(rStripped)) return r.getId();
}

// Tier 3: Contains match
for (NovelContinentRegion r : regions) {
    if (r.getName().contains(name) || name.contains(r.getName())) return r.getId();
}

    ```java
    public static <T extends NamedEntity> Long matchByName(List<T> candidates, String targetName, List<String> suffixes)
    ```
    - Null/empty targetName or null/empty candidates -> return null
    - Tier 1: Loop candidates, exact string match with targetName
    - Tier 2: If suffixes is non-null/non-empty, stripSuffix from targetName and from each candidate name. If stripped target is non-empty and equals stripped candidate -> return that ID
    - Tier 3: Loop candidates, check if candidateName.contains(targetName) (when targetName.length >= 2) OR targetName.contains(candidateName) (when candidateName.length >= 2). Log at WARN level when Tier 3 match succeeds.
    - If all tiers fail, log WARN and return null

    **4. stripSuffix helper:**
    ```java
    public static String stripSuffix(String name, List<String> suffixes)
    ```
    - Null/empty name -> return ""
    - Sort suffixes by length descending (greedy match)
    - If name ends with a suffix, strip it and return the remainder
    - If no suffix matches, return original name

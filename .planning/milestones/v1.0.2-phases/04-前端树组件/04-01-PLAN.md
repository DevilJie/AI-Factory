**Use a recursive render function approach.** Define the node rendering as part of the main template, and for children, just nest the same `v-for` pattern. Since Vue 3 `<script setup>` supports self-reference by filename, create the component to accept EITHER top-level `nodes` prop OR internal `treeData`:

```typescript
// Props definition
const props = withDefaults(defineProps<{

  disabled: boolean
  nodes?: Faction[]  // for recursive self-call; undefined for root
```


When rendering child `<FactionTree>`, bubble events:
